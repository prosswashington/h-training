 Create new project and configure suite-link clients and DA Server

In Derivation view right-click on \$h_Area and select New -> Instance. Label the Project Area as (Training_PLC01)).

In Derivation view right-click on \$h_ABSuiteLinkClient and select New -> Instance. Label the Project Suite-Link as (Training_SL_PLC01).

Switch back to Deployment View. By default these newly created instances will be in the Unassigned Host folder. Select and drag these items within
the (TrainingTagEngine) instance. Next, referring to the images below configure the Areas and Suite-Links communication path.

The DA Server, located in the SMC (System Management Console) is the middle man between the HMI and the PLC program.
The first image points the DA Server to the PLC program by its IP address. The second image points the DA Servers to the Areas created in ArchestrA.

 Troubleshooting using the SMC

The SMC also provides other essential tools for development. In the first image the log viewer is used to identify any
performance warnings or errors within ArchestrA. In the second image the Platform Manager is used to check a status of
platform such as Running on Scan or offline.

 Creating Galaxy backups

Galaxy Backups is a very important part of the development process. Backups should be made daily when there is heavy HMI development.
Within the SMC -> Galaxy Database Manager right-click on the HPN_Olathe_Training select Backup. Depending on the size of the galaxy a backup
should around 10 minutes.

Create a new HMI screen and add graphics based on project drawings
  • To create a new screen Right-click on HMI Screens and select New -> Symbol.
  • Rename the new symbol to the screen name “s_TrainingWindow”
  • Open the new screen and begin added graphics followed by inserting tag references when needs. Graphics are located in “HPN Graphics”.
  • Once screen development is complete, open “\$Training_ViewApp”. This will open WindowMaker.

The next step is to reference is the new screen within the HMI application. Right-Click in the “Windows” box and select “New”. Using the Window
properties in the image below, insert the sane credentials. Once complete this will bring up a blank screen. In the Toolbar above click on the
Embedded ArchestrA Graphic icon and import the entire symbol that is the screen titled “S_TrainingWindow”. Once complete save and close WindowMaker.
